# Changelog

## 2026-06-08

### 実装内容

- 総合ランキングを3-2-1点方式へ変更。
- 総合点が同点の場合はカタン合計点で順位判定するように変更。
- 総合ランキングに総合点、カタン合計点、1位/2位/3位回数を表示。

### 技術的変更

- `games` / `players` の保存形式は変更せず、既存データを新ルールで再計算する方式を維持。
- Androidアップデート時のデータ引き継ぎのため、`android.package` は変更しない方針を明記。

## 2026-05-30 - Ver1.3

### 実装内容

- 固定3人の開拓者による戦績管理MVPを完成。
- 試合結果登録、編集、削除を実装。
- ポイント入力から順位を自動判定する方式へ変更。
- 同順位に対応。
- 総合ランキングを実装。
- 全期間 / 今月 / 先月 / 任意期間の期間集計を実装。
- 現在の島王カードを追加。
- 総合ランキング称号を追加。
- 試合履歴称号を追加。
- 戦績リセットを追加。
- CSVエクスポートを追加。
- 開拓者名変更を設定画面へ移動。
- Android APK向けEAS Build設定を追加。

### UI変更

- 淡い羊皮紙・砂色ベースのデザインへ調整。
- 背景色を黒くならないようlightテーマ固定へ調整。
- Web / Androidでスクロールできるようレイアウトを修正。
- 総合ランキングと履歴の順位バッジを金・銀・銅系に色分け。
- ヒーローカード、島王カード、ランキング称号、空状態にイラスト素材を追加。
- イラスト画像の透明余白をトリミング。
- アプリアイコン、スプラッシュ画像、Android adaptive iconを設定。

### 不具合修正

- React Native WebでView直下にテキストノードが入る問題を修正。
- 総合ランキングが表示されない問題を修正。
- 完全同点時に `0位` が表示される問題を修正。
- Webで削除確認 / 戦績リセット確認が動作しない問題を修正。
- 背景色修正後にスクロールできない問題を修正。
- ポイント順位整合性を手動チェックする方式を廃止し、ポイントから順位を自動計算する方式に修正。

### 技術的変更

- AsyncStorageを利用した `games` / `players` の分離保存。
- `playerId` を安定IDとして保存し、表示時に現在の開拓者名を参照。
- `src/utils/ranking.ts` に順位計算・ランキング計算を集約。
- `src/utils/titles.ts` に称号計算・称号画像選択を集約。
- `src/utils/csv.ts` にCSV出力処理を分離。
- `src/utils/confirm.ts` でWeb / Native両対応の確認処理を実装。
- `src/constants/images.ts` でイラスト画像参照を一元管理。
- TypeScript strict、`npm run typecheck`、`npm run lint` を運用。
- Expo SDK 56 / React Native 0.85.3 / React 19.2.3で動作。
