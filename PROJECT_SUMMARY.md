# Project Summary

## アプリ概要

「島の記録帳」は、兄弟3人で遊ぶカタン風ボードゲームの試合結果を記録するローカル専用の戦績管理アプリ。
試合日と3人分のポイントを入力すると、順位を自動判定し、履歴・3-2-1点方式の総合ランキング・称号・CSV出力へ反映する。

## ターゲットユーザー

- 自分
- 兄
- 弟

主利用者は開発者本人。Android端末での個人利用とAPK配布を想定している。

## 解決する課題

- 紙やメモで散らばりやすいカタン風ゲームの戦績を一元管理する。
- 開拓者名を後から変更しても、過去の履歴やランキングに現在名を反映する。
- 試合後にポイントだけ入力すれば順位を自動判定できるようにする。
- 総合ランキングや称号で、長期的な戦績を楽しく見返せるようにする。
- 総合ランキングは1位=3点、2位=2点、3位=1点で集計し、同点時はカタン合計点で判定する。
- CSV出力でバックアップや表計算ソフトでの確認を可能にする。
